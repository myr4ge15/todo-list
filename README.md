# todo-list

Простой веб-трекер личных задач с жизненным циклом: бэклог → в работе → готово / отмена.
Ванильный JS + HTML + CSS, без сборки. Данные хранятся в localStorage браузера.

**Live-версия:** https://myr4ge15.github.io/todo-list/

## Возможности

- Добавление задач и переключение статусов (бэклог ⇄ в работе, завершение, отмена/возврат).
- Итог по завершённой задаче (короткая заметка, ≤50 симв.).
- Фильтры: все / бэклог / в работе / готово / отмена.
- Редактирование текста задачи двойным кликом.
- Экспорт и импорт задач в JSON (бэкап и перенос между устройствами).
- **PWA:** устанавливается на телефон и десктоп, работает офлайн.

## Установка как приложение (PWA)

Открыть [live-версию](https://myr4ge15.github.io/todo-list/) и:

- **Chrome / Edge (десктоп):** значок «Установить» в адресной строке.
- **Android (Chrome):** меню ⋮ → «Установить приложение».
- **iOS (Safari):** «Поделиться» → «На экран Домой».

После установки приложение запускается в своём окне и работает без интернета (данные хранятся локально).

## Запуск

Открыть `index.html` в браузере — сборка не нужна.
Примечание: офлайн-режим (service worker) активен только по HTTPS или на `localhost`, не при открытии через `file://`.
